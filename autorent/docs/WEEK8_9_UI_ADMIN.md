# 8-апта және 9-апта нәтижесі

## 8-апта: UI/UX + Backend API интеграция

Орындалды:

- Іздеу/сүзгілеу кеңейтілді:
  - Автопарк: іздеу, статус, санат, баға аралығы, сұрыптау.
  - Админ rental timeline: `user_id`, `car_id`, `status_filter`.
  - Админ client requests: статус бойынша сүзгі.
- Мобиль UX:
  - Фильтр блогы мобильде `toggle` арқылы ашылады/жабылады.
- Empty-state/UI feedback:
  - Бос тізімдерге хабарламалар қосылды.
  - Жүктелу және қате хабарламалары көрсетіледі.

Интеграция:

- Фильтр параметрлері backend API query-ге тікелей байланған.
- `Promise.all` арқылы админ беті бірнеше endpoint-ті біркелкі жүктейді.

## 9-апта: Админ-панель

Орындалды:

- Админ workspace жаңа бөліммен толықты:
  - `Пользователи / Қолданушылар`.
- Пайдаланушыларды басқару:
  - `/admin/users` тізімі.
  - Іздеу: `ID/email/аты`.
  - Рөл, верификация күйін көрсету.
  - `/admin/transfer/{user_id}` арқылы admin рөлін басқа verified user-ге беру.

Тексеру:

1. `cd autorent/autorent-pro/frontend`
2. `npm run build`
3. `npm run dev`
