# Contributing

## Local setup

Backend:
1. `pip install -r requirements.txt`
2. `python -m uvicorn app.main:app --reload`

Frontend:
1. `cd autorent-pro/frontend`
2. `npm install`
3. `npm run dev`

## Code style

Backend:
- Format with `black`
- Sort imports with `isort`
- Lint with `ruff`

Frontend:
- Format with `prettier`

## Branching

Follow `docs/BRANCHING.md`.
